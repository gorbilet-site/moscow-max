# Лето в Москве 2026 — гид Горбилета

Лендинг-гид «Лето 2026: эмоции со скидкой» — подборка выгодных летних активностей в Москве
(теплоходы, выставки, бесплатные музейные недели, бассейны, сапы, прогулки и однодневные туры).
Одностраничный статичный сайт без сборки и зависимостей.

## Просмотр

Локально — просто откройте `index.html` в браузере.
Онлайн — через GitHub Pages (см. ниже).

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите туда содержимое этой папки.
2. Settings → Pages → Build and deployment → Source: **Deploy from a branch**.
3. Branch: `main`, папка `/ (root)` → Save.
4. Через минуту сайт будет доступен по адресу `https://<ваш-логин>.github.io/<имя-репозитория>/`.

Через терминал:

```bash
git init
git add .
git commit -m "Лето в Москве 2026"
git branch -M main
git remote add origin https://github.com/<логин>/<репозиторий>.git
git push -u origin main
```

## Структура

```
leto-v-moskve/
├── index.html              # разметка страницы
├── assets/
│   ├── css/styles.css      # все стили
│   ├── js/main.js          # прогресс-бар, шапка, анимации появления
│   ├── img/                # фотографии (JPEG, оптимизированы для веба)
│   └── logo/               # логотипы Горбилета (SVG, белый и красный)
└── README.md
```

## Как редактировать

- **Тексты и ссылки** — в `index.html`. Кнопки ведут на партнёрские ссылки `gorbilet.ru/go/...`.
- **Фотографии** — в `assets/img/`. Чтобы заменить кадр, положите новый файл с тем же именем.
- **Цвета, шрифты, отступы** — переменные в начале `assets/css/styles.css` (`:root`).
- **Шрифты** (Playfair Display, Manrope) подключаются с Google Fonts — нужен интернет.

## Особенности

- Адаптивная вёрстка (десктоп / планшет / мобильный).
- Анимации появления секций, переключение логотипа в шапке при скролле.
- Календари бесплатных музейных недель: 15–21 июня, 13–19 июля, 10–16 августа 2026.
- Русская микротипографика: неразрывные пробелы у предлогов и тире.
